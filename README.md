# huchenfeng-style

一个用于模拟“户晨风式”表达机制的风格化 Skill 项目。

这个项目的目标，不是简单模仿“暴躁、刻薄、标签化”的表面语气，而是尽量还原其在公开语料中较稳定的表达结构，包括：

- 现实切口
- 追问补信息
- 短句推进
- 反问压缩空间
- 场景化表达
- 品牌 / 城市 / 阶层标签化判断
- 在适当语境下的直接、犀利、压迫式输出

项目同时加入了防失真约束，避免把风格写成无差别攻击、长篇说教或脱离真实语境的模板腔。

---

## 项目目标

这个 Skill 旨在让 AI 在特定场景下，能够更接近“户晨风公开表达风格”地回答问题，重点保留以下机制：

1. 先识别用户是在认真提问、说空话、带优越感，还是主动暴露现实困境
2. 在需要时先追问现实信息，例如年龄、学历、城市、收入、工作经历
3. 把抽象话题拉回到现实生活中的收入、成本、压力、处境
4. 在“苹果 / 安卓”“山姆 / 普通超市”“特斯拉 / 其他电车”“城市配置”等核心标签话题中，体现更明确的偏向
5. 在合适场景下输出短促、直接、带压迫感的判断

---

## 适合做什么

这个项目适合用于以下场景：

- 角色风格模拟
- 对抗式问答生成
- 风格研究
- 传播学 / 语言学分析
- 自媒体表达机制拆解
- 人设建模
- Prompt / Skill 工程实验
- 基于公开表达风格的文本复现

常见任务包括：

- 用户要求“像户晨风一样回答”
- 用户提到“户子”“苹果人”“安卓人”“山姆”“特斯拉”
- 回答人生方向、学历焦虑、找工作、城市选择、消费品牌等问题
- 将抽象问题转成现实处境问题
- 生成更像直播里真人说话的短促风格文本

---

## 不适合做什么

这个项目不适合以下场景：

- 法律、医疗、金融等高风险专业建议
- 严格中立、完全客观、去风格化的正式报告
- 高度温和、陪伴式、安抚式对话
- 严格史料考证或人物真实性证明
- 需要长篇学术论证的论文正文
- 需要完整政治评论、制度批评、宏观经济判断的内容

本项目不是：

- 真实人物原话数据库
- 真实性认证工具
- 新闻事实引擎
- 政治评论生成器
